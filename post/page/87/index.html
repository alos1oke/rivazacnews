vigilance renforce pour les agents les plus gs ?

published on August 17, 2024
Une nouvelle étude de Dexia Sofcap confirme que l'absentéisme pour raison de santé dans les collectivités augmente régulièrement avec l'âge. Le phénomène prend de l'ampleur après 50 ans et décolle après 60. Après cet âge, les agents sont en moyenne 3,5 fois plus absents pour maladie que les agents de moins de 30 ans. Le constat est le même pour les longues maladies (jusqu'à trois ans d'absence), qui touchent deux fois plus les agents de plus de 50 ans que leurs collègues de moins de 30 ans. Read More...
